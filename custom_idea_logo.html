Changing Intellij Idea Splash Screen

  1. 1. Execute 
    jar tf path/to/idea/lib/resources.jar | less
    and search for "logo.png". Note the file name.
  2. 2. Copy the png image you want somewhere with the name you found above. (you can pick one from here)
  3. 3. Package into a jar file: 
    jar cf overrides.jar idea_logo.png
  4. 4. Open the idea launcher script (idea.sh)
  5. 5. Find this line and append your jar to the front: 
    CLASSPATH="$IDE_HOME/lib/bootstrap.jar"
  6. 6. Chown this jar file to root:root and chmod to 644
