Modifying Intellij Idea without using SDK or compiling it from source

  • intellij

You can modify the behavior of Intellij IDEA using a mechanism similar to videogame patching. This guide shows how to change the startup splash screen.

  1. Execute jar tf path/to/idea/lib/resources.jar | less and search for "logo.png". Note the file name.
  2. Copy the png image you want in some directory with the name you found above. (you can pick one from here)
  3. Open the idea launcher script (idea.sh) in an editor
  4. Find this line and append your directory to the front: CLASSPATH="$IDE_HOME/lib/bootstrap.jar"
  5. Chown this directory to root:root and chmod to 755

Now the chosen splash screen will be shown on startup. This method can be used to override classfiles as well. Extract a classfile from lib, decompile it (https://jdec.app), modify it, compile it (javac -cp "path/to/idea/lib/*" Main.java) and place it a subdirectory corresponding to the package. Intellij is a huge project (multi million lines of code), so compiling it is a no go if you don't have a beefy machine. This is a patching mechanism, not a plugin, so you can change the ide in any way you want, however the changes won't be portable.
